# St. Kizito Parish App - Project Requirements ## Project Identity - **Name:** St. Kizito Parish App - **Type:** Catholic liturgical + parish community mobile app - **Platforms:** iOS & Android (Native feel) - **Target:** Local parish (Nigeria) + global Catholics - **Vibe:** Peaceful, sacred, focused, warm candlelight, quiet dignity. (Ref: Calm, Headspace, Apple Health). ## Design System ### Colors - **Light:** #F9F6F0 (Warm parchment) / #1A1A1A (Charcoal text) - **Dark:** #0F1117 (Navy-black) / #F0EBE0 (Cream text) - **Sepia:** #F5ECD7 / #3B2A1A - **High Contrast:** #FFFFFF / #000000 - **Liturgical Accents:** - Ordinary Time: #4A7C59 (Green) - Advent/Lent: #6B4E8A (Purple) - Christmas/Easter: #C9A84C (Gold) - Pentecost/Martyrs: #B5303C (Red) - Marian: #3A6EA5 (Blue) ### Typography - **Headings:** Georgia / Humanist Serif - **Body:** System Sans (SF/Roboto) - Min 16px, 18px default. - **Liturgical Text:** Elegant Serif. - **Readings Line Height:** 1.7–1.9. ### Layout & Navigation - **Margins:** 24px - **Radius:** 16px - **Touch Targets:** 48x48px min. - **Navigation:** Bottom Tab Bar ([Home] [Readings] [Divine Office] [Parish] [More]). ## Core Screens 1. **Home:** Today's liturgical summary, Hero reading card, Quick access row, Parish pulse. 2. **Daily Readings:** Date nav, text size controls, Drop-caps, liturgical accents. 3. **Divine Office:** Time-aware hub, 2x3 prayer card grid. 4. **Liturgical Calendar:** Month view with season-colored dots, detail panel. 5. **Mass Booking:** Multi-step flow (Intention -> Summary -> Processing -> Confirmation). 6. **Parish Hub:** Tabs for Announcements, Events, Gallery, Mass Times. 7. **Settings:** Theme switcher (4 tiles), Reading prefs, Notifications. 8. **Favourites:** Tabbed (Readings, Verses, Prayers). 9. **Jobs Board:** Professional card list and details. 10. **Daily Inspiration:** Featured verse card + supporting reflections. ## Accessibility - WCAG AA (4.5:1 min contrast). - Scalable text (up to 24px+). - Screen reader support. - No color-only information.
